Les balises de CSS que nous avons appris

Balise HTML Description Exemples
color Couleur du texte. color:red;
font-size Taille des caractères. font-size:20px; /*on peut utiliser les unités: cm in pt em ex ...*/
font-style Style des caractères: bold, italic, underline. font-style:bold;
text-align Aligner: center, left, right. text-align:center;
background-color Couleur de fond. background-color:black;
background-image Image de fond. background-image:url("images/Philosophes.jpg")
background-repeat Répétition de l'image de fond background-repeat:no-repeat;/*repeat-x,repeat-y*/
background-position Position de l'image de fond *background-position:30px 50px;
background-attachement Adhérence de l'image de fond background-attachement:fixed; /*scroll*/
background Propriété raccourcie permettant de remplacer les cinq propriétés que l'on vient d'aborder ci-dessus, vous pouvez écrire les valeurs des cinq propriétés dans l'ordre que vous souhaitez. background: #9f9f9f url('images/entete.png') bottom right repeat-x; background:url("images/Philosophes.jpg") no-repeat top right fixed;
background:-moz-linear-gradient(left top,red,blue,green,yellow);
-moz-border-radius:10px;

width Largeur. width:800px;
height Hauteur height:200px;
font-family Spécifier le font d'un paragraphe. font-family:"Comic Sans MS";
text-indent Indentation de la première ligne. text-indent:30px;
font-variant Capitalisations des caractères. font-variant:small-caps;
text-transform Changer la capitalisations. text-transform:uppercase; /*lowercase,capitalize,none*/
text-shadow Ombre du text. text-shadow:3px 3px 5px white;
border Bordure: on doit avoir 3 propriétés: largeur, type de la bordure, couleur de la bordure pour que la bordure puisse apparaitre. border:none;
border: 1px solid black;
border-top
border-top:5px dashed #66ff66;
align
align:center;
